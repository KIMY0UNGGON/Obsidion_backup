$\sigma(x) = \begin{cases} \infty & \text{if } x = 0 \\ 0 & \text{if} x \neq 0 \end{cases}$ 


 가우시안의 분포는 분산이 0에 가까우면, 평균지점에서 무한히 좁아지고 높이는 무한히 높아지는 스파이크(spike) 모양으로 변함.

이와 같은 가우시안 분포는 다음과 같은 식으로도 표현 가능.
$lim_{\delta->0}N(y|\mu,\delta^2) -> \delta(y-\mu)$

 이때 위의 그림을 참고해서 보자면 x가 0일때 y축의 값이 무한대에 수렴하고,
  0이 아닐때 y축이 0으로 수렴한다. 이를 수식으로 표현하면:
  
  $\sigma(x) = \begin{cases} \infty & \text{if } x = 0 \\ 0 & \text{if} x \neq 0 \end{cases}$ 

이렇게 되고, 이를 디랙 델타 함수(dirac delta function)라고 한다.

델타함수의 특징은 **한점에서만 값이 무한대**를 가진다는 점이다.
다른 위치에선 0의 값을 갖는다.

당연히 정규분포 모양이므로 디랙델타의 전체 pdf, 전체 넓이는 1이다.

$\int_{-\infty}^{\infty}\delta(x) = 1$

이런 delta 함수를 변형하면 다음과 같은 수식을 만들 수 있다.

$\delta(x-y) = \delta_y(x) = \begin{cases} \infty & \text{if } x = y \\ 1 & \text{if} x \neq y \end{cases}$


 이때 $x-y$가 0이어야 delta함수가 $\infty$값을 가지니까 $x= y$일때만 delta함수가 $\infty$가 됨.
 

$\int_{-\infty}^{\infty}f(y)\delta(x-y)dy = f(x)$

위의 식은 선별 속성(sifting property)라고 하며 디랙 델타 함수를 사용해서 정의한다.
위 식을 보자면 델타 함수 $\delta(x-y)$가 있는데 이 함수는 $x=y$가 아닐때는 모두 0이므로 
$x=y$일때만 적분이 성립, 즉 단 한지점의 값만 고려한다.
$x=y$이므로 $f(y)*1 = f(x)$가 되는 것을 보인다. 
	이때 F(t)가 아닌이유는 적분의 특성 때문이다. 현재 적분식은 $-\infty$~$\infty$까지 즉 모든 구간의 값을 더하는 것이다.
	현재 $x=y$인 구간을 제외한 모든 구간이 0이므로 적분을 $f(x)*1$ 만이 남게되어 $f(x)$가 된다. 


선별 속성이 쓰이는 곳과 만들어지게 된 이유:
	신호 처리에서 아날로그 소리를 디지털 데이터로 바꾸는 샘플링 과정을 수학적으로 이상화하게 된 것이 선별 속성이다.
	연속적인 신호(아날로그)에서 특정 시점(x)의 값 (f(x))을 완벽하게 추출하기 위해 사용된다.
	
   공학에서는 시스템의 순간적인 입력에 어떻게 반응하는지 분석하는데 쓰인다.

   확률분포에서는 모든 확률이 단 하나의 값에 집중되어 있음을 나타낼때 사용한다.
   선별속성은 연속적인 셰게에서 하나의 이산적인 점을 다루기 위한 수학적 도구이다.
