# MLP(Multi-Layer Perceptron) 다층 퍼셉트론

  MLP는 엄밀히 말하면 다층 피드포워드 신경망.

 다층 피드포워드 신경망(Multi-Layer Feedforward Neural Network)
 MLP는 회귀 문제를 풀때에도 사용됨. 그 이유는 관습적으로 계속 불러왔기 때문.

 MLP에서 회귀인지 아닌지는 출력층에서 결정됨.

### 분류 => 결과값이 0~1사이의 값이 나옴

### 회귀 => 결과값이 $-\infty$ ~ $\infty$사이의 값이 나옴


 출력층의 Activation Function이  Sigmoid나 Softmax 같은 0~1사이의 값이 나오는 함수면 분류모델이 됨.

 만약 선형 함수 같은 경우는 회귀 모델이 됨.
